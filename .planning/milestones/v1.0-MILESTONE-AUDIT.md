### Integration Gap

**RESET does not clean IndexedDB images** (medium severity)

- `PlottedApp.tsx` dispatches `{ type: "RESET" }` which returns `initialState` (empty `maps` array)
- `deleteMapImage` is only called in `MapHeader.tsx` (individual map delete)
- After RESET, orphaned image blobs remain in IndexedDB permanently
- **Fix:** Call `deleteMapImage(map.id)` for each map before dispatching RESET
- **Affected:** DATA-01, SETUP-03
